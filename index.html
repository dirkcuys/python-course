When does it start?

The course will start at the end of April. If time permits join us then, but if that is not the case, don't fret. The course is built in a way that it can be self-paced, taken at any given time.

How long will it take?

The class will last for six weeks. However if you want, you can always slow down or speed up. The materials are there to be used and you can use them however you want.

How will this all go down?

Every week there will be email fired your way, to introduce the materials for the week and gently remind you to work and collaborate with others in the community.

What will I learn?

You will learn the basic use of setting up a Python environment and get to know building blocks of programing, such as Data structures, functions, classes, etc. In the end we will build a nice little project, which will tie all the knots together.

Can I do it?

Why not? This course is aimed at complete beginners. In fact, you will probably meet people who will need your help during this course. All you need is a computer, internet access and thirst for knowledge. Be brave!

Why should I?

Why should you not? Coding is the new black, so it does not hurt to know what all the fuss is truly about. In the end, you will see it is a lot of fun. And who knows, maybe this is a first step towards a new beginning for you.
